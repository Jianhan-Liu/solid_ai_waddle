2. 预训练模型的目的
    - 在庞大的无标注数据上进行预训练可以获取**更通用的语言表示**，并有利于下游任务
    - 为模型提供了一个**更好的初始化参数**，在目标任务上具备更好的泛化性能、并加速收敛
    - 是一种**有效的正则化手段**，避免在小数据集上过拟合（一个随机初始化的深层模型容易对小数据集过拟合）
